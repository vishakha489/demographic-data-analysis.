# Demographic Data Analysis

## Overview
This project analyzes demographic data from the 1994 U.S. Census dataset using **Python** and **Pandas**. The dataset includes features such as age, workclass, education, occupation, race, sex, hours worked per week, and salary. The goal is to extract insights about income distribution, education, occupation, and working patterns across different demographic groups.

The analysis addresses questions including:

- Number of people by race.
- Average age of men.
- Percentage of people with a Bachelor's degree.
- Income distribution based on advanced education levels.
- Minimum working hours and income analysis.
- Country with the highest percentage of high-income earners.
- Most popular occupation among high-income earners in India.

The dataset used in this project is sourced from the **UCI Machine Learning Repository**:

**Reference:**
